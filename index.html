Gadget Recommendation System


The Gadget Recommendation System leverages a Generative AI model, along with Retrieval-Augmented Generation (RAG) and Large Language Models (LLMs) to provide personalized device suggestions based on user preferences. Users input their preferences, such as budget, desired features, and brand preferences, through an intuitive interface. The system then uses RAG to retrieve relevant device data from a large database, which is processed by an LLM to generate a clear, insightful recommendation with a rationale behind each choice. The system can recommend devices like smartphones, laptops, and other gadgets, explaining why specific models are the best fit for the userâ€™s needs. Powered by Django for backend management and a dynamic database, the system continuously improves its recommendations through user feedback, ensuring a highly personalized experience.
