# Personal Portfolio Website

This is a personal portfolio website built using **HTML** and **CSS**, showcasing my professional background, skills, and projects. It includes sections for my profile, skills, projects, and contact information, allowing visitors to learn more about me and get in touch.

## Features

- **Profile Section**: Introduction and career summary.
- **Skills Section**: Overview of technical skills and tools I have worked with.
- **Projects Section**: Display of key projects with descriptions and technologies used.
- **Contact Section**: Links to my **LinkedIn** and **Email** for professional connections.

## Technologies Used

- **HTML**: Used for structuring the content of the website.
- **CSS**: Used for styling and layout of the website.
- **Responsive Design**: Ensures the website is mobile-friendly and adapts to various screen sizes.
